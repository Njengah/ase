# Snake Game Tutorial Prompts (HTML, CSS, JavaScript)  
------------------------------------------------------------------------------

## 1. Start  

**Prompt:**  

Let’s create a classic **Snake Game** using **HTML, CSS, and JavaScript**. The game will include smooth movement, food spawning, collision detection, and score tracking. Ready to begin?  

------------------------------------------------------------------------------

## 2. Setting Up the Environment  

**Prompt:**  

We need an `index.html`, `style.css`, and `script.js` file. How do we set up the basic project structure and link the files together?  

------------------------------------------------------------------------------

## 3. Creating the Game Board  

**Prompt:**  

Let’s create a game board using an **HTML canvas**. Generate the necessary code to set up the canvas, define its size, and style it using CSS.  

------------------------------------------------------------------------------

## 4. Drawing the Snake  

**Prompt:**  

How do we draw the snake on the canvas using JavaScript? Generate code to represent the snake as a series of squares.  

------------------------------------------------------------------------------

## 5. Adding Snake Movement  

**Prompt:**  

Let’s implement keyboard controls. How do we move the snake using the arrow keys and update its position on the canvas?  

------------------------------------------------------------------------------

## 6. Spawning Food  

**Prompt:**  

Generate code to place food at random positions on the board. The snake should be able to eat it and grow.  

------------------------------------------------------------------------------

## 7. Growing the Snake  

**Prompt:**  

How do we make the snake’s body grow when it eats food? Store the snake’s body segments in an array and update it dynamically.  

------------------------------------------------------------------------------

## 8. Handling Collisions  

**Prompt:**  

Let’s add collision detection. How do we end the game if the snake collides with the walls or itself?  

------------------------------------------------------------------------------

## 9. Displaying the Score  

**Prompt:**  

How can we display and update the player’s score based on the number of food items eaten?  

------------------------------------------------------------------------------

## 10. Game Over Logic  

**Prompt:**  

Create a game-over screen. How do we allow the player to restart by pressing the spacebar?  

------------------------------------------------------------------------------

## 11. Adding Sound Effects  

**Prompt:**  

Integrate sound effects for eating food and when the game is over. How do we load and play audio files using JavaScript?  

------------------------------------------------------------------------------

## 12. Enhancing with DeepSeek AI  

**Prompt:**  

How can we use **DeepSeek AI** to make the game smarter? Suggestions: AI-controlled snake, predictive movement, or an AI opponent.  

------------------------------------------------------------------------------

## 13. Additional Features  

**Prompt:**  

What improvements can we add? Ideas: increasing speed over time, power-ups, obstacles, or a high-score leaderboard.
